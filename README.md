# RiskVector.app - Global Emergency Intelligence Platform

A real-time risk monitoring platform for travelers.

## Architecture

- **Frontend**: Next.js 14 (App Router, React Server Components)
- **Backend**: Next.js API Routes
- **Maps**: Leaflet (Open Source)
- **Styling**: Tailwind CSS
- **Real-time**: Server-Sent Events (SSE)
- **Data Sources**: GDELT, WHO, Weather APIs, Government Alert Systems

## Quick Start

```bash
npm install
npm run dev
```

## Features

- Real-time risk alerts
- Interactive risk map
- Multi-source data aggregation
- Location-based notifications
- Mobile-responsive design

## Data Sources

1. GDELT Project (Political events, conflicts)
2. OpenWeather (Weather alerts)
3. USGS (Earthquakes)
4. WHO/CDC (Health alerts)
5. Government Alert Systems (KATWARN, WEA, etc.)
